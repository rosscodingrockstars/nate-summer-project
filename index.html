Counting Money

You will be given a random target number at the start of the game. There are four currencies below. Clicking on a currency will add a specific amount of points to your total user score. You win the game by matching your total user score to the target number. You lose the game if your total score goes above the target number. The value of each currency is hidden until you click on it. Each time the game starts, the values of each currency and the target number will change.
